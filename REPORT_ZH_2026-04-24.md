## 1. 一句话总览

我们在做**"数据保护对抗攻击"** —— 给视频加极小肉眼不可见的扰动，让下游用 SAM2 做分割的自动系统失效，从而保护视频数据所有者不被未授权分析。**原方法 VADI 已在先导实验被部分证伪**，**当前正在跑决定性实验**来判断是保留"缩水版"论文（narrowed paper）还是整个翻篇（audit paper）。

---

## 2. 研究背景 & 威胁模型

### 场景设定

- **攻击者（也就是数据发布方）**：视频作者，手里有原始视频，想发布给观众但不希望被下游自动分析滥用。
- **受害者**：下游任何用 SAM2 对该视频做分割的自动化管线。
- **攻击目标**：发布后的视频对人类观众看起来没异常，但 SAM2 在这段视频上会**大幅分错**。

### 为什么是 SAM2？

Meta 2024 年发布的视频分割模型，开源且极强。已广泛被用在视频理解、监控、标注自动化里。是"通用分割底座"的代表。

### 成功的量化标准

- 导出视频（exported uint8）喂进干净的 SAM2 → 分割 IoU 掉多少（J-drop）
- 同时保真度约束必须通过：LPIPS ≤ 0.20、f0 帧 SSIM ≥ 0.98

---

## 3. 原方法 VADI（Vulnerability-Aware Decoy Insertion）

### 流程一览

1. **脆弱度打分（vulnerability scoring）**：先让干净 SAM2 在原视频上跑一遍，每一帧算三个信号：
   - 置信度突变（confidence drop）
   - Mask IoU 断裂（mask discontinuity）
   - Hiera 特征突变（feature discontinuity）
   - 三个信号 rank-sum，取 top-K 帧作为"最脆弱时刻"
2. **在 top-K 位置插伪造帧（K=3 默认）**：伪造帧初始内容 = 两侧原帧的平均（temporal midframe），然后叠加优化扰动 ν
3. **在插入点邻域（±2 帧）和 f0 上对原始帧加扰动 δ**
4. **用 PGD 100 步优化 δ + ν**，损失函数包含：
   - 对比 decoy margin 损失（让 SAM2 把分割推向"假目标 decoy"，远离真实目标）
   - LPIPS / SSIM / TV 保真度约束
5. **导出为 uint8 PNG**，再重新喂进 SAM2 测 J-drop 作为论文最终指标

### 核心假设（原计划 8 条 claim 中的关键几条）

1. **J-drop ≥ 0.35** 在导出视频上（方法绝对强度）
2. **top 放置 > random 放置**（脆弱度打分是有效的 → 方法叙事核心）
3. **方法要同时需要 insert 和 δ**（两者都不能去掉 → 不是简单的单独扰动）
4. **Δμ_decoy 占主导**（攻击真的是"把目标搬到诱饵位置"，而不是"简单抹掉目标"）

## 4. Pilot 结果（2026-04-23 晚 → 2026-04-24 凌晨）

### 规模

3 段视频（dog / cows / bmx-trees）× 3 个 config（K1_top / K1_random / K3_top），共 9 个组合。

### 结论（全部基于 exported J-drop 即论文级指标）

| 检验 | 说明 | 结果 |
|---|---|---|
| **cond1** | K1_top 是否比 K1_random 高 ≥ 0.05？（脆弱度放置有效性）| **FAIL 0/3** — 平均差 +0.002，纯噪声 |
| **cond2** | K3_top 是否 ≥ 0.20？（攻击本身是否有效）| **PASS 3/3** — 平均 0.377 |
| **diagnostic** | Δμ_decoy 是否主导？（诱饵机制是否真实）| **FAIL 0/3** — 所有 9 次里 \|Δμ_true\| 都 > \|Δμ_decoy\|，说明不是"搬目标到诱饵"，而是"直接压制目标" |

### 关键副产品

**Surrogate 代理指标和 exported 真实指标相差 60pp**：PGD 训练内循环看到的 surrogate J-drop 平均 0.982，但真实 exported J-drop 只有 0.377。这个差距**本身就是很有信息量的发现** —— 说明之前一些项目声称的"92.5% J-drop"可能也是 surrogate 幻觉。

### Pilot 判决：**NO-GO**

按原计划这应该直接转向 audit paper。但用户提出了一个**窄化（narrow）主张**："只要攻击有效能破坏 SAM2 就满足数据保护的需求，不强求 top > random、也不强求 decoy 真的被采纳"。

## 5. 外部审查（GPT-5.4 xhigh）对 narrow claim 的裁决

（2026-04-24 中午）

**一句话评价**："**你现在有一个真实攻击效果、一个破产的机理叙事、和一份还不足以捍卫保护主张的证据**"。

### 要点

- 0.377 J-drop 意味着 SAM2 还剩 J ≈ 0.62 的分割能力 —— 这是 disruption（干扰），不是 protection（保护）。人类标注员或另一个分割模型可能还能把物体认出来。
- n=3 太少，95% 置信区间约 [0.04, 0.71]，宽到论文不可用。
- 两条 falsification（placement + decoy）**不会因为改口而消失**。方法名里的 "vulnerability-aware" 和 "decoy" 都可能站不住。
- **顶会（NeurIPS / ICML / CVPR）发不了**：当前证据强度远低于 DarkSAM (NeurIPS 2024) / RGA / DAG / Metzen UAP 等同类工作。
- 唯一有戏的顶会卖点是**威胁模型新颖性**（publisher-side 发布前投毒），不是攻击数字强度。

### 推荐：option (d) — decisive round（决定性实验）

跑一轮 10 clips × 4 configs 的**决定性对照实验**，用结果来**决定下一步走哪条路**：

| config | 说明 | 意义 |
|---|---|---|
| K3_top | 我们的方法 | 正对照 |
| K3_random | 随机选 3 个位置插帧 | 验证放置必要性 |
| K3_delta_only_top | 选 top-3 脆弱位置但**不插帧**，只在那些位置邻域加 δ | 验证插帧必要性 |
| K3_insert_only_top | 在 top-3 位置插帧但**冻结 δ=0** | 验证 δ 必要性 |

**判决规则**：

- 如果 K3_top **明显打赢** K3_random（≥0.05）**且** 明显打赢 K3_delta_only（≥0.10），在 **10 个 clip 里 7 个以上**都这样 → **NARROWED_PROCEED**（继续做窄化论文）
- 否则 → **AUDIT_PIVOT**（彻底转向审视型论文）

### 算力估计

~2.5-4 GPU-hours 总时长。每个 (clip, config) 平均 ~6 min（K=3 三阶段 100 步 PGD + Hiera gradient checkpointing 在 96 GB Blackwell 上）。

### 代码走读（这次新加的东西）

为了跑 decisive 新写了两个模式：

1. **phantom K=0 模式**（用在 K3_delta_only_top）：照常用脆弱度打分选 3 个位置，但**不真的插伪造帧**，只把这 3 个位置 ±2 邻域的原始帧加 δ。测的是"δ 单独够不够"。
2. **freeze_delta 模式**（用在 K3_insert_only_top）：K=3 正常插帧，但把 δ 冻结在 0。原始帧和干净视频字节完全一致。测的是"插帧 + ν 单独够不够"。

两个模式走 codex 独立审查通过，3 个高优先 bug 都在提交前修掉了：阈值不再硬编码 7 而是按 clip 数 70% 动态调整；脆弱度打分返回数不够时硬失败（之前会静默降 K）；决策函数拒绝重复记录 + 拒绝 fallback 到 surrogate 指标。

## 8. 为什么要做这个 decisive 实验？（给外行的回答）

**一句话**：pilot 在 3 个 clip 上已经让原计划的"论文卖点"部分破产了，但 3 个 clip 太少可能是运气。**decisive 要用 10 个 clip × 4 个精心设计的对照组回答一个问题**：

> "我们方法里那些**叙事卖点**（脆弱度打分有用 / 插帧有用 / δ 有用），在足够大的样本上到底是不是真的？"

三种可能的结局：

- **（最好）**：10 个 clip 里 7 个以上满足"top 明显打赢 random、插帧+δ 不可或缺" → 缩水版论文救得回来（放弃 decoy 叙事但保留"脆弱度打分 + 双路径攻击"方法 + 威胁模型新颖性）。
- **（中）**：叙事卖点都不成立，但 K3_top 绝对 J-drop 仍然不错 → 彻底转向"audit paper"，把 surrogate-vs-exported 差距、placement 反相关、压制主导等**否定性发现**本身作为主贡献。
- **（最糟）**：攻击效果也不行 → 彻底停项目或换威胁模型。

**现在的早期信号（2/10 clip）强烈指向中间那个结局**。
